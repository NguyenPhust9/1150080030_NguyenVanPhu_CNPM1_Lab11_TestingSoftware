# Selenium Framework

## 📌 Giới thiệu

Đây là project automation test sử dụng Selenium để kiểm thử web tự động.

## 🛠 Công nghệ sử dụng

* Java
* Selenium WebDriver
* TestNG (hoặc JUnit)
* Maven

## ▶️ Cách chạy project local

### 1. Clone project

git clone https://github.com/<your-username>/selenium-framework.git

### 2. Mở project bằng IntelliJ hoặc Eclipse

### 3. Cài dependencies

mvn clean install

### 4. Chạy test

mvn test

## 📂 Cấu trúc project

* src/test/java: chứa test cases
* src/main/java: chứa code chính
* pom.xml: cấu hình Maven
